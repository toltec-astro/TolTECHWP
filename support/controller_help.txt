== Help information for controller ==

Enter commands here: Format is recipient (agent) name 
    followed by space then by the command to send.
    
    To send command to the same agent, omit the agent
    name but press <return> twice.

Special Commands:
    help: displays this message
    help all: will request help messages from all agents
    exit: will quit the controller application
    status: will list all available recipients (agents)
    config: will print the current configuration
    
Remember: Only one response is returned per command.
    Press <CR> on an empty line to get outstanding responses.
